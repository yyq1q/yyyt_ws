# ラズベリーパイマウス周りの作成したコンポーネント
障害物除去のため作っていたが時間が無かったため，障害物除去を行わないことになった．
そのためこれらのコンポーネントは使用していない．
# raspai2shougaibutsu
ラズベリーパイマウスの障害物センサーを用いた障害物除去コンポーネント
# controller2raspai
指令振り分けコンポーネント
# 動作環境
OpenRTM

# 使用手順
Controllerコンポーネントとcontroller2raspaiを接続，
